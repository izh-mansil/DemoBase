Обработка предназначена для анализа строковых констант в текстах запросов.

Работает с файловыми базами, серверными базами и выгруженными файлами XML.

 

Для работы с файловой базой необходимо заполнить следующие параметры и настройки:

* Путь к базе - строка подключения к текущей базе.

* Пользователь - пользователь под которым запущено предприятие.

* Пароль - пароль к текущей базе.

* Каталог хранилища - строка подключения к каталогу хранилища базы.

* Пользователь хранилища - пользователь хранилища для текущей базы.

* Пароль хранилища - пароль хранилища текущей базы.

 

Для работы с серверной базой необходимо заполнить следующие параметры и настройки:

* Кластер серверов 1с: Предприятия - кластер серверов текущей базы.

* Имя информационной базы в кластере - имя текущей базы в кластере серверов.

* Пользователь - пароль к текущей базе.

* Пароль - пароль к текущей базе.

* Каталог хранилища - строка подключения к каталогу хранилища базы.

* Пользователь хранилища - пользователь хранилища для текущей базы.

* Пароль хранилища - пароль хранилища текущей базы.

 

Для работы с выгрузкой XML необходимо заполнить следующие параметры и настройки:

* Каталог выгрузки файлов XML - путь, по которому расположены XML файлы выгрузки конфигурации.

 

Содержит команды:

- Выполнить анализ строковых констант текстах запросов - при нажатии происходит анализ текстов запросов на наличие ключевых слов запроса в строковых константах. После выполнения процедуры, происходит открытие дополнительной формы с результатами выполнения анализа, которая содержит команды:

- Выполнить автозамену в таблице - при нажатии в выбранных строках происходит преобразование всех символов ключевых слов в нижний регистр. Также замену можно произвести в ручном режиме, выбрав нужную строку в колонке "Строка для замены".

- Поместить изменения в файлы и загрузить в конфигурацию - при нажатии происходит запись в файлах, отмеченных флажками, на основании данных табличной части на формы и загрузка этих файлов в конфигурацию.

Анализирует строковые константы текстов запросов, находя ключевые слова запроса внутри текстовых констант и преобразует их в нижний регистр.
